# Решение проблем с отображением

## Если ничего не отображается:

1. **Откройте консоль браузера** (F12 или Cmd+Option+I на Mac)
   - Проверьте, есть ли ошибки JavaScript (красные сообщения)
   - Скопируйте ошибки и сообщите о них

2. **Проверьте, что бэкенд запущен:**
   ```bash
   # В папке app
   python -m uvicorn app.main:app --reload
   ```
   Бэкенд должен быть доступен на `http://127.0.0.1:8000`

3. **Проверьте, что фронтенд запущен:**
   ```bash
   # В папке shop
   npm run dev
   ```
   Фронтенд должен быть доступен на `http://localhost:5173`

4. **Если видите ошибку "Failed to load products":**
   - Это означает, что React работает, но API не отвечает
   - Убедитесь, что бэкенд запущен
   - Проверьте CORS настройки в бэкенде

5. **Если страница полностью белая:**
   - Откройте консоль браузера и проверьте ошибки
   - Возможно, проблема с Tailwind CSS
   - Попробуйте обновить страницу (Cmd+R или Ctrl+R)

6. **Переустановите зависимости:**
   ```bash
   cd shop
   rm -rf node_modules package-lock.json
   npm install
   npm run dev
   ```

## Что должно отображаться:

- Навигационная панель сверху с "Shop", "Products", "Cart"
- Заголовок "Products"
- Поле поиска
- Панель фильтров слева
- Список товаров (или сообщение об ошибке, если API не работает)
