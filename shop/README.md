# Shop Frontend

React + TypeScript + Vite приложение для интернет-магазина.

## Установка и запуск

### 1. Установите зависимости

```bash
cd shop
npm install
```

Это установит все необходимые пакеты, включая:
- react-router-dom (для роутинга)
- tailwindcss, postcss, autoprefixer (для стилей)
- react-toastify (для уведомлений)
- axios (для API запросов)

### 2. Запустите dev сервер

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

### 3. Убедитесь, что бэкенд запущен

Фронтенд ожидает, что бэкенд API работает на `http://127.0.0.1:8000/api`

Если бэкенд на другом адресе, измените `baseURL` в файле `src/services/api.ts`

## Структура приложения

- **Главная страница** (`/`) - список товаров с фильтрами, поиском и пагинацией
- **Страница товара** (`/product/:id`) - детальная информация о товаре
- **Корзина** (`/cart`) - просмотр и управление корзиной (изменение количества, удаление товаров)

## Функционал

- ✅ Поиск товаров
- ✅ Фильтрация по категории, цене
- ✅ Сортировка по цене и названию
- ✅ Пагинация
- ✅ Добавление товаров в корзину
- ✅ Управление корзиной (изменение количества, удаление)
- ✅ Уведомления об успешных/неуспешных операциях
