# Инструкция по запуску фронтенда

## Установка зависимостей

Выполните в терминале в папке `shop`:

```bash
npm install
```

Это установит:
- react-router-dom (для роутинга)
- tailwindcss, postcss, autoprefixer (для стилей)
- остальные зависимости

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Структура приложения

- **Главная страница** (`/`) - список товаров с фильтрами и поиском
- **Страница товара** (`/product/:id`) - детальная информация о товаре
- **Корзина** (`/cart`) - просмотр и управление корзиной

## API

Фронтенд ожидает, что бэкенд запущен на `http://127.0.0.1:8000/api`

Если бэкенд на другом адресе, измените `baseURL` в файле `src/services/api.ts`
